# Expo + Nativewind Wallet

## Features

- Built with Expo and Nativewind
- Dark/light mode
- TypeScript 

## Getting Started

```bash
# Use Node.js v20
nvm use 20

# Install dependencies
npm install

# Start the Expo development server with a clean cache
npx expo start -c
```
